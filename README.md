# 结算系统

打开 `index.html` 后输入结算月份、结算总额和结算时间，系统会自动计算：

- 运营团队：结算总额 × 5%
- 市场团队：结算总额 × 10%
- 合计支出：结算总额 × 15%
- 净额：结算总额 - 合计支出
- KK：净额 × 25%
- K：净额 × 7%
- T/A：净额 × 34%

当前规则按整数向下取整。页面默认不填案例数据，输入结算月份、结算总额和结算时间后自动生成结果。手机打开时会自动使用 H5 单列布局和底部操作栏。

比例配置在 `index.html` 的 `CONFIG` 里。如后续团队比例或股东比例变动，只需要改对应数组。

## 本地服务器

在本目录运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\start-server.ps1
```

然后访问 `http://localhost:8787/`。
